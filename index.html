How It Works

API to skills in seconds

Skillify reads your API surface, understands its structure, and generates skills that any OpenClaw-compatible agent can use immediately.

1

Discover

Point skillify at an OpenAPI spec, GraphQL schema, HTML docs, or Python package. It extracts every endpoint automatically.

↓
2

Analyze & Group

Endpoints are grouped into logical skills by resource and domain. An editable JSON spec lets you fine-tune groupings before generation.

↓
3

Generate

Each group becomes a skill directory with SKILL.md, references, and auth scripts. Ready to drop into your agent's workspace.

↓
4

Install & Use

Skills install directly into nanobot with --install. Your agent can immediately call any API endpoint through natural conversation.

Supported Sources

Works with what you have

No matter how your API is documented, skillify can work with it.

OpenAPI / Swagger

Full support for OpenAPI 3.x and Swagger 2.0 specs. Deterministic parsing, no LLM needed.

--source-type openapi

GraphQL

Introspect any GraphQL endpoint or parse .graphql schema files directly.

--source-type graphql

HTML / Markdown Docs

AI-powered extraction from any documentation page. Just point it at the URL.

--source-type html

Python Packages

Discover API surfaces from installed Python packages via introspection.

--source-type python
Built For Agents

Skills that just work

🔐

Secure by Default

API keys are stored locally and loaded at runtime. Secrets are never hardcoded in skill files.

📦

Progressive Disclosure

Large APIs use reference files to keep skill context lean. Agents load details only when needed.

⚙️

Key Management

Built-in skillify keys commands to store, list, and manage API keys for generated skills.

📝

Editable Specs

The intermediate JSON spec is human-editable. Rename skills, adjust groups, or remove endpoints before generating.

🤖

Self-Hosted Skill

Skillify ships as a nanobot skill itself. Generate new skills from inside your agent via natural language.

⚡

Any LLM

Powered by litellm. Use Anthropic, OpenAI, OpenRouter, or any compatible provider.

Get Started

Install in seconds

Install with pip

$ pip install openclaw-skillify

Requires Python 3.11+

Or install with uv

$ uv tool install openclaw-skillify

Installs as a standalone CLI tool

Then generate your first skills

$ skillify run https://api.example.com/openapi.json --install

Discovers, groups, generates, and installs to nanobot in one step

Or use inside nanobot

$ cp -r skill/skillify ~/.nanobot/workspace/skills/

Then ask your agent: "Turn the Stripe API into skills"
